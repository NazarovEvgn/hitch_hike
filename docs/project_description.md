# Lets-app описание

- Приложение Lets позволяет удобно находить нужный сервис (Парикмахерская, Автомойка, Ателье и т.п.) и записываться онлайн на услугу.
- Отличительной чертой приложения является возможность бизнесу выставить статус "Свободны. Готовы принять". Этот инструмент позволяет привлекать заказы в периоды низкой загруженности или отстутствия заказов.
- Приложением пользуется две стороны: Бизнес (он же владелец сервиса) и Клиент (consumer).
- Обе стороны используют приложение преимущественно со смартфона. Также есть версия PWA для использования через браузер.
- Путь клиента: Зайти в прлиожение, на карте или через продвинутую систему поиска найти подходящий сервис. В карточке сервиса оформить запись-онлайн на интересующую услугу.
- Путь бизнеса: зайти в приложение и оформить подписку. Затем зарегистрировать сервис (или несколько сервисов). заполнить всю необходимую информацию о сервисе. Далее работать с приложением через админ панель: получать онлайн-записи, а также объявлять акции, услуги и цены и пользоваться инструментом объявления статуса "Свободны" при необходимости.

## Цель проекта

- Ключевая цель - создать систему онлайн-записи, которая будет лучше коннкурентов на рынке. Все лучшие практики существующих + прокачка того что хочет бизнес и 
пользователь.
- Приложение должно быть самым удобным способом для поиска сервиса и процесса онлайн-записи.
